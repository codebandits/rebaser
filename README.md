## Installation

Add the [JitPack](https://jitpack.io/) Maven repository to your list of repositories:

```
repositories {
  ...
  maven { url 'https://jitpack.io' }
}
```

Add the compile dependency:

```
dependencies {
  ...
  compile 'com.github.codebandits:rebaser:master-SNAPSHOT'
}
```
